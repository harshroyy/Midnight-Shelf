**Midnight Shelf** is a dedicated horror-genre book store built to bring the darkest tales to your doorstep. This full-stack e-commerce application offers a spine-chilling browsing experience, allowing users to unearth, collect, and purchase the scariest literature available.

Built using the **MERN** stack (MongoDB, Express, React, Node.js), the application features secure authentication, a "haunted" cart system, and a robust admin dashboard to manage the inventory from the shadows.
